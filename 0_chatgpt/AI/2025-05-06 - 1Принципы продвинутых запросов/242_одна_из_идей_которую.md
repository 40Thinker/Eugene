# Wiki: “Cognitive Overload as Neuro-Trophic Adaptation”  
**TL;DR:** Instead of copying elite training plans, **implant the *thinking patterns*** of elite disciplines (math, philosophy, systems biology, linguistics) into the athlete’s daily environment. Package them as micro-doses that *feel* like drills but re-wire decision loops. The athlete becomes a self-evolving AGI-like operator, not a data-driven puppet.

---

## 1. Industry Default → My Rejection  
**Default engineer view:**  
“Collect HRV, power, lactate → feed to ML → output optimal plan.”  
**My rejection:**  
That pipeline ignores *how* the athlete *thinks*. When the model is wrong the human has no meta-tools to notice or fix it.  
→ Need to train the *update-function* itself, not only the parameters.

---

## 2. Paradigm Shift: Import Thinking Patterns, not Data  
Elite disciplines (topology, phenomenology, game theory) have **mature, high-resolution inference rules**.  
I propose to **graft those rules onto sport** instead of scraping more biological data.  
Example: use variational-calculus language to *feel* the minimal-energy path inside your own gait.  
→ The knowledge is exact; the sport becomes the *laboratory* for that knowledge.

---

## 3. Why Athletes Reject It (and the Wrapper Solution)  
Problem: pure math or Husserl triggers “academic pain” → psychophysiological shut-down.  
Solution: **micro-dosed, emotionally-anchored wrappers**  
- 3-minute “topology” drill hidden as *“find the shortest *felt* line during 200 m”*  
- English vocabulary game framed as *“train the ligaments of signal-processing”*  
→ Load is real, wrapper is playful, rejection drops.

---

## 4. From Reps to Combinatorics  
Old: repeat set → recover → repeat.  
New: **combinatorial task space**  
- Same workout, but athlete must choose among 5 conflicting data streams (RPE, HRV, coach cue, competitor, weather) and *justify the choice in real time*.  
Synapses strengthen via **cognitive super-compensation**, not only muscular.

---

## 5. Measurable Cognitive Hypertrophy  
We track:  
- latency of internal model update (seconds between surprise and new tactic)  
- false-positive rate of “I’m empty” calls  
- cross-domain metaphor generation (can the athlete explain today’s set with chess, music, war?)  
→ These KPIs proxy **neuro-trophic adaptation** better than VO₂-max curves.
